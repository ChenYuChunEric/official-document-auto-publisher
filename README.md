# 公文改名稱及發布自動化工具

> [!IMPORTANT]
> **本專案目前僅適用於「臺北市南港區胡適國民小學」公版網站。** 若其他學校欲使用此工具，必須修改程式碼中對應的 XPath 設定，以符合該校網站的排版結構。

這是一個專為學校（如臺北市立胡適國民小學）設計的公文自動化處理與發布工具。它能自動從原始 PDF 檔案中擷取主旨與內文，重新命名檔案，並透過網頁自動化工具（Selenium）將公文發布到指定的網站區塊。

## 🌟 主要功能

- **自動解壓縮**：自動處理來源資料夾中的 ZIP 壓縮檔。
- **PDF 文字擷取**：使用 PyMuPDF 自動解析公文 PDF，精準提取「主旨」與「說明」內容。
- **智能清理**：自動移除公文中的雜項資訊（如檔號、保存年限、頁碼等），讓公告內容更乾淨。
- **檔案自動組織**：
  - 支援「西元年」或「民國年」日期前綴選擇。
  - 將公文及其附件自動分類至以標題命名的資料夾中。
- **分類介面**：提供直覺的 GUI 介面，讓使用者快速將公文分類至「宣導與公告」、「榮譽榜」、「競賽與活動資訊」或「研習進修」等區塊。
- **網頁自動發布**：
  - 整合 Selenium 自動化操作瀏覽器。
  - 支援臺北市單一身分驗證（LDAP）登入流程。
  - 自動填寫標題、內容並上傳多個附件。

## 🛠️ 技術棧

- **語言**：Python 3.x
- **GUI 框架**：Tkinter (內建)
- **PDF 處理**：PyMuPDF (fitz)
- **網頁自動化**：Selenium
- **瀏覽器驅動**：ChromeDriver (需與 Chrome 瀏覽器版本對應)

## 📦 安裝需求

在使用此工具前，請確保已安裝以下 Python 套件：

```bash
pip install pymupdf selenium
```

此外，您的電腦需具備：
1. **Google Chrome 瀏覽器**。
2. **ChromeDriver** (Selenium 會自動嘗試管理，或請確保其在系統路徑中)。

## 🚀 使用說明

1. **執行程式**：
   - 直接執行 `公文改名稱及發布.py` 或使用已打包的 `公文改名稱及發布.exe`。
2. **選擇路徑**：
   - 程式啟動後，請先選擇「野生的來文資料」資料夾（來源）。
   - 接著選擇「可以被看懂的來文資料」資料夾（輸出）。
3. **選擇日期格式**：
   - 選擇要在資料夾名稱前增加的日期格式（西元、民國或不增加）。
4. **公文分類**：
   - 程式會逐一顯示擷取到的公文標題，請點擊介面上的按鈕進行分類或選擇「跳過」。
5. **網頁發布**：
   - 分類完成後，程式會自動開啟瀏覽器導向學校網站。
   - 請依照提示在網頁上完成手動登入動作。
   - 點擊「確認登入」後，程式將開始自動填寫資料並上傳附件。

## ⚠️ 注意事項

- **XPath 設定**：網頁發布功能高度依賴網頁結構，若網站改版，可能需要更新程式內的 `announcement_button_xpaths` 等相關路徑設定。
- **登入驗證**：由於安全機制，登入過程（含單一身分驗證）需由使用者手動完成。
- **檔案權限**：請確保對所選資料夾具有讀寫權限。
